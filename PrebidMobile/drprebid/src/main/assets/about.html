Dr. Prebid

Dr. Prebid

Version: 1.0

Dr. Prebid is here to help you validate and troubleshoot your implementation of Prebid Mobile. You can use this app to validate your settings for your ad server setup, Prebid Server configuration, and end-to-end Prebid Mobile SDK implementation. You can also use Dr. Prebid to troubleshoot in cases where you’re not receiving the bid responses you’re expecting.

Simply enter the information about your configuration into the Setup screen and Dr. Prebid will run three tests to validate your configuration:

  • Ad Server Setup Validation

    Determines whether your line items are configured correctly on your ad server.

  • Real-Time Demand Validation

    Validates your Prebid Server setup. This test validates that the real-time requests generated using your Prebid Server Account ID and Prebid Server Configuration ID are reliably returning demand.

  • End-to-End SDK Validation

    An end-to-end test validating your complete configuration. In this test, Prebid Mobile SDK communicates with Prebid Server (using your account and configuration details) and with the primary ad server SDK.

For complete documentation for Dr. Prebid, including details on interpreting validation test results, see Dr. Prebid App on Prebid.org.
